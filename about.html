About Botanica

A digital apothecary bridging ancient botanical wisdom with modern web technology.

Our Purpose

Welcome to Botanica, a comprehensive digital catalogue dedicated to medicinal plants and herbs. Our platform allows users to explore a rich database of nature's remedies, complete with detailed plant profiles, medicinal uses, active chemical compounds, and preparation methods. Whether you are a researcher, a student, or a natural health enthusiast, Botanica provides a structured, easy-to-navigate interface to discover the botanical world.

Why We Created This Website

Botanica was developed as a group project for the TWT2231 – Web Techniques and Applications course. Our objective was to design and implement a fully functional Medicinal Plants and Herbs Database System that applies core web technologies including HTML5, CSS3, JavaScript, PHP, and MySQL.

Project Scope & Features

This academic project features two main modules built with fundamental web techniques:

  • Public Guest Module: A user-friendly interface allowing anyone to browse, search (by common or botanical name), and filter plants by family or medicinal uses without registration.
  • Detailed Plant Profiles: Comprehensive pages displaying habitat origins, dosages, and safety precautions.
  • Administrator Module: A secure, session-managed backend for administrators to perform full CRUD (Create, Read, Update, Delete) operations on the plant inventory, complete with image uploads.
  • Security Focus: Implementation of secure, parameterized MySQL queries and dual-layer (client & server) form validation.
