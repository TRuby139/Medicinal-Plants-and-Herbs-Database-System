# Web Tech Project: Medicinal Plants and Herbs Database System

This project is a web-based database system for medicinal plants and herbs, built using standard HTML5, CSS3, and JavaScript.

## Project Progress

Here is a simple summary of what we have built in the recent updates:

1. **Global Design System**: Set up the main CSS (styles) and JavaScript (interactivity) files. We created a "Modern Apothecary" theme with green and parchment colors that looks professional but relies entirely on vanilla web code (no external frameworks).
2. **Public Guest Module**: Built the `index.html` (home page catalogue) and `plant-detail.html` (detailed plant view) for public visitors to search and view herbs.
3. **Administrator Module**: Built the `login.html` and `admin-dashboard.html` pages for administrators. The dashboard includes a pop-up form (modal) to add or edit plant records, complete with image previewing.
4. **Final Polish & Organization**: Fixed mobile screen layouts so the website works well on phones, improved the form behaviors, and organized the project folders so that assignment guidelines and non-code files are ignored by GitHub.
